Project 1
再次實現：
    爲了讓程序有更好的可擴展性，需要使其更加模塊化。以下為可能的模塊：
	1.  分析器：讀取文本，管理其他類；
	2.    規則：檢查塊的類型並進行相應的格式化；
	3.  過濾器：包裝處理内嵌元素的正則表達式；
	4.處理程序：分析器可使用它產生輸出。

    首先是處理程序：
        HTMLRender類為文本添加標簽，
    規則：
        用於識別塊並對其進行轉換，所以包含兩個方法：condition和action；
            condition需要一個參數，返回布爾值表示規則是否適用於指定的塊；
            action用來訪問處理程序對象。
        規則的超類則包含一些共用的處理程序方法，如：start(), feed(), end()等。
    分析器：
        其核心是Parser類，利用處理程序、規則和過濾器來將文本轉化爲HTML文件。
